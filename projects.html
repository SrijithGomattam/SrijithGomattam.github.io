Weekly Planner

  • Implemented features to create tasks and events, as well as options to save, delete, and customize calendar entries.
  • Employed JavaFX, FXML, and Java in the IntelliJ IDE to create a visually engaging and functional interface.
  • Utilized file I/O for data persistence, enabling users to save and load their calendar data effectively.
  • Conducted unit testing with JUnit to ensure reliability and bug-free functionality.
  • Integrated multiple object-oriented design patterns, showcasing advanced Java programming techniques.

Stock Analysis

Conducted a Stock Analysis project focused on analyzing stock market trends and predicting future movements. This data-driven project utilized Python and a variety of data science libraries, reinforcing skills in exploratory data analysis (EDA) and predictive modeling. Key aspects include:

  • Fetched stock market data using the Yahoo Finance API, allowing for comprehensive data collection and organization.
  • Performed Exploratory Data Analysis (EDA) using Pandas, Numpy, Matplotlib, and Seaborn to uncover patterns and insights.
  • Built classification and regression models using scikit-learn to predict stock price movement, enabling informed decision-making.
  • Visualized data trends and analysis results through graphs and charts, making complex data accessible and interpretable.
